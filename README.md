# 📊 Super Store Sales Dashboard (Power BI)

## 🔍 Project Overview

This project presents an interactive Power BI dashboard built to analyze Super Store sales data. It provides insights into sales performance, profit trends, and customer segmentation to support data-driven decision-making.

---

## 🚀 Key Features

* 📈 Sales analysis by region, category, and segment
* 💰 Profit trend and performance tracking
* 🎯 Customer segmentation insights
* 🔄 Interactive filters and slicers for dynamic exploration

---

## 🛠 Tools & Technologies

* Power BI
* Excel / CSV Dataset

* Technology category contributed the most profit
* High discounts negatively impact profit margins
